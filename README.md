# vant-vue-fm

## 简介

基于 vue 的开的单页面应用 —— 音乐app。

项目在开发中···

使用到的技术：

- vue（框架）
- vue-cli（脚手架）
- vue-router（路由）
- vuex（状态管理）
- vant （UI组件库）
- less（css 预处理语言）
